YT-Media-Downloader

A lightweight desktop utility built with Python and Tkinter that downloads audio or video directly from YouTube using yt-dlp and FFmpeg.
Designed to be simple, fast, and dependency-minimal.

No browser automation.
No cookies required.
No unnecessary complexity.

Just paste → choose format → download.

FEATURES : 
• Download Audio (MP3)
• Download Video (MP4)
• Separate folders for Audio and Video
• Real-time progress bar
• Clean minimal GUI
• Uses yt-dlp internally (reliable + actively maintained)
• Works without cookies for public videos
• Single file script – easy to run anywhere

Requirements: 
• Python 3.11+
• yt-dlp
• FFmpeg

step 1:
Install yt-dlp:
pip install yt-dlp

step 2: 
Install ffmpeg
  step 1 : winget install "FFmpeg (Essentials Build)"
  step 2 : check using "ffmpeg -version"
  step 3 : Get path "where ffmpeg" and copy it
  step 4 : update the path in command list
  step 5 : Add the FFMPEG location path in the env
